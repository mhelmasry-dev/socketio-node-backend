# 🔔 Real-Time Backend — Socket.io & Node.js

A real-time backend server built with Node.js, Express, and Socket.io supporting live notifications and event-based communication.

## 🔗 Links
- **GitHub:** [https://github.com/mhelmasry-dev/socketio-node-backend](https://github.com/mhelmasry-dev/socketio-node-backend)

---

## ✨ Features

- ⚡ **Real-time Communication** — Bidirectional events between client and server
- 🔔 **Live Notifications** — Instant push notifications
- 🗄️ **MongoDB Integration** — Persistent data storage


---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Node.js | Runtime |
| Express.js | Web Framework |
| Socket.io | Real-time Communication |
| MongoDB + Mongoose | Database |


---

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/mhelmasry-dev/socketio-node-backend.git

# Navigate to the project
cd socketio-node-backend

# Install dependencies
npm install

# Create .env file and add your variables
cp .env.example .env

# Start the server
npm start
```

### Environment Variables

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```
